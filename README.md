# 🚁 Helipad Marker Detection

Project phát hiện và nhận diện **helipad marker (ký hiệu bãi đáp trực thăng)** sử dụng Computer Vision / AI.

---

## 📌 Giới thiệu

Dự án này nhằm:
- Phát hiện vị trí helipad trong ảnh hoặc video
- Hỗ trợ các hệ thống UAV / drone
- Ứng dụng trong điều hướng và hạ cánh tự động

---

## 🧠 Công nghệ sử dụng

- Python
- OpenCV
- YOLO / TensorFlow / PyTorch
- NumPy
